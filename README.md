# FACTURACION FACIL

Este repositorio contiene una aplicacion para ayudar con la contabilidad de una empresa pequeña o mediana.

# SET UP DEL REPOSITORIO

Asegurate de tener instalado node js y npm, despues, ejecuta el setup y escoje la version de archivo dependiendo tu sistema opertivo.

# DEPENDENCIAS

Deberias tener instalado edge para windows y google-chrome o brave para linux.
